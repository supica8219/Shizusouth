# Shizusouth

静岡理工科大学周辺の通学を想定し、バスおよび電車の時刻情報をシンプルに閲覧できるWebアプリケーションです。  
※本サービスは現在公開を終了しています。(稼働時期：2021年4月~2026年3月) 

---

## 概要

通学時に必要なスクールバスおよび鉄道の時刻情報を、素早く確認できることを目的として開発しました。

特定の駅や利用シーンにフォーカスし、必要な情報をシンプルに表示する設計としています。

## 📊 運用実績

- 運用期間：2021年4月〜2026年3月
- 月間PV：約1000〜3000
- 利用シーン：通学時の時刻確認
---

## 主な機能

- スクールバスの時刻情報表示
- 駅単位での時刻情報表示
- シンプルかつ高速なUI
- モバイル端末での利用を考慮した設計

---

## 技術構成

- フロントエンド: HTML / CSS / JavaScript
- バックエンド: PHP

---

## 開発背景

大学周辺の通学において、複数のサービスを横断して時刻を確認する必要がある課題があり、それを一元化する目的で開発しました。

実際の利用シーンを想定し、必要な情報へ最短でアクセスできる構成を意識しています。

---

## 公開終了について

本サービスは一定期間公開・運用を行いましたが、データの取り扱いや継続的な運用面を踏まえ、現在は公開を終了しています。

本リポジトリでは、アプリケーションの設計および実装の参考としてコードを公開しています。  
なお、一部のデータおよび取得処理については含めていません。

---

## 工夫した点・学び

- 利用シーン（通学）に特化したUI設計
- 情報への到達時間を短縮するための画面設計
- 実運用を通じた改善（ユーザー利用を前提とした開発経験）
